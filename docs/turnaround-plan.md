# 株式会社WADAYA 再建案(叩き台 v0.1)

> ⚠️ 財務数値がない状態での**仮説込みの叩き台**です。「🔍要確認」の項目はヒアリング用の質問を兼ねます。
> 前提とした経営課題: ①中大規模撮影・eラーニング受託のコモディティ化 ②商品開発プロセスの機能不全 ③LED背景への無計画な投資

---

## 1. 現状認識 — 何が起きているか

3つの課題は独立した問題ではなく、ひとつの構造である。
**「請負制作」というビジネスモデル自体が賞味期限を迎えた。**

- **中大規模撮影のコモディティ化** — 機材の低価格化、フリーランスの増加、生成AIによる映像制作の民主化。発注側は「そこそこの品質なら安く」を選べるようになり、価格決定権を失った。
- **eラーニングのコモディティ化** — 撮影より深刻。スライド+AI音声+自動編集ツールで、企業が「内製」できる領域に変わった。受託で戦う市場ではなくなりつつある。
- **LED投資の失敗** — 投資自体より「商品が先になく、設備が先にあった」ことが本質的な失敗。ただし、**サンクコストとして見れば、今や同業が持っていない差別化資産**でもある。問題は稼働率と回収設計だけ。
- **商品開発の機能不全** — 上記すべての根因。「作れるものを作る」発想で、「売れるものを検証してから作る」プロセスがなかった。

再建の核心は、個々の事業の立て直しではなく、**「請負」から「資産を貸す・教える・伴走する」モデルへの転換**。

## 2. 再建の基本方針 — 3本柱

### 柱① LEDスタジオを「コスト」から「商品」に変える(止血兼成長)

LEDウォールは制作案件の道具として持つには重すぎるが、**時間貸し商品**としては在庫ゼロで売れる。

- **スタジオレンタル(素貸し+オペレーター付きの2段階料金)** — バーチャルプロダクション対応スタジオは首都圏でもまだ希少。横浜立地は都内より価格優位を出せる
- **配信・対談の定額スタジオ契約** — 月次ウェビナーを打つ企業に「毎月◯時間+配信オペ込み」のサブスク。請負の単発売上をストック売上に変える最短ルート
- **KPIはひとつ、稼働率** — 現状の月間稼働時間を即計測し、損益分岐となる稼働率を算出するのがPhase 1の最初の仕事

### 柱② eラーニング受託 → 「内製化支援」へピボット(コモディティ化の波に乗る側へ)

内製化の波と戦わず、**波を売る**側に回る。葉加瀬アカデミーでの映像教育・ワークショップ・バーチャルプロダクション教育システム導入支援という「教える」実績は、制作会社には稀有な、複製されにくい資産。

- **企業向け「動画内製化支援プログラム」** — 研修+体制構築コンサル+機材選定+「難所だけ制作受託」のハイブリッド。顧客が内製化するほどWADAYAの研修・伴走収益が増える構造
- **バーチャルプロダクション人材研修** — スタジオ(柱①)を教室として使う。設備と教育資産の掛け算で、稼働率と単価を同時に押し上げる
- 受託eラーニング制作は新規営業を止め、既存顧客の維持のみ(撤退ではなく縮小固定)

### 柱③ 撮影事業は「LEDでしか撮れない案件」に絞る(選択と集中)

中大規模撮影を全方位で受けるのをやめ、**バーチャルプロダクションが価格比較を無効化する案件**に集中する。天候リスクが許されない、ロケ不可、多シーンを1日で撮る、海外風景が必要——といった案件では、LEDは「贅沢」ではなく「合理的な選択」として売れる。相見積もりの土俵から降りることが脱コモディティの実体。

## 3. 商品開発の立て直し — 仕組みの導入

LED投資の失敗を繰り返さないためのルール3点セット:

1. **「売ってから作る」** — 新商品は仮価格でLP・提案書を作り、既存顧客5社に当てて受注(または強い購入意思)が取れてから開発投資する
2. **全商品に固定価格パッケージを置く** — 「お見積りします」だけの商品は比較検討の入口に立てない。松竹梅の価格表を公開する(`brand-content-draft.md` のFAQ整備と直結)
3. **3ヶ月検証・撤退基準の事前設定** — 「受注◯件未満なら畳む」を始める前に書面化する

## 4. フェーズ計画

| フェーズ | 期間 | やること | 成否の指標 |
|---|---|---|---|
| **0. 診断** | 〜1ヶ月 | 財務実態の把握(§6)、スタジオ稼働率の計測開始、顧客別粗利の棚卸し | 数字が揃うこと |
| **1. 止血** | 〜3ヶ月 | スタジオレンタル開始、赤字案件・低粗利顧客の整理、固定費圧縮、受託eラーニング新規停止 | 月次キャッシュフローの黒字化 |
| **2. 転換** | 3〜12ヶ月 | 内製化支援プログラムと定額配信契約の販売開始、撮影の案件選別運用、Webサイトを新商品中心に再構築 | ストック型売上が月商の20% |
| **3. 確立** | 12ヶ月〜 | 研修事業のスケール(教材化・オンライン化)、スタジオ稼働率の安定運用 | ストック型売上が月商の40%、稼働率◯%以上 |

進行中のサイトリニューアルは **Phase 2 の販売装置**として位置づけ直す。現状のコンテンツ素案は「制作会社の信頼訴求」が中心だが、再建方針が固まれば「スタジオレンタル」「内製化支援」の独立した商品ページ(価格表つき)が最優先コンテンツになる。

## 5. やめることリスト

再建では「始めること」より重要。

- 価格勝負になる中大規模撮影の相見積もり案件への参加
- 受託eラーニング制作の新規営業
- 商品設計のない設備・機材投資(§3のルール3点セットを通らない投資はすべて)
- 「何でもできます」型の総合訴求(6領域横並びの見せ方は商品の輪郭を消す)

## 6. 判断に必要な情報 🔍要確認

1. **資金繰りの残存月数**(現預金÷月次バーンレート)— Phase 1の過激さが決まる。借入返済が重い場合はリスケ交渉も選択肢
2. **LED投資の残債と償却状況** — リース or 購入か、月の固定負担額
3. **顧客別・案件種別の粗利** — 「中大規模撮影」が本当に儲かっていなかったのかの事実確認。売上が大きいだけで粗利率最低、が典型パターン
4. **スタジオの現在の月間稼働時間**
5. **教育事業の現収益と先方(葉加瀬アカデミー等)との契約形態** — 柱②の展開自由度に直結

## 要約

LEDは「失敗した投資」ではなく「商品設計されていない資産」と捉え直して時間貸し・教室として即収益化し、eラーニングは受託から内製化支援へ反転させ、撮影はLEDが効く案件だけに絞る。**「請負」から「貸す・教える・伴走する」への転換**が再建の軸。

§6の数字が揃い次第、損益シミュレーション付きの v0.2 に更新する。
